# Vercel Deployment

Deploy this project to Vercel:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/AnkithSVaidya/EcoStellar)

## Quick Deploy Steps:

1. Push this repo to GitHub (if not already)
2. Go to https://vercel.com
3. Click "Import Project"
4. Select this repository
5. Vercel will auto-detect Vite and deploy

## Or use CLI:

```bash
npm i -g vercel
cd frontend
vercel --prod
```
